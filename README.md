- 📧 Email reminders for upcoming study blocks

**Frontend**:  
- React  
- TailwindCSS

**Backend**:  
- Django REST Framework  
- PostgreSQL

**DevOps**:  
- Docker & Docker Compose  
- GitHub Actions  
- Deployed on Render / Fly.io
